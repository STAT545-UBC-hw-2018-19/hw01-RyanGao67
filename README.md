# Code sample
```Python
class Solution:
    def multiply(self, num1, num2):
        len1 = len(num1)
	len2 = len(num2)
	result = 0
	for i in range(len1):
	    for j in range(len2):
	        nu1 = ord(num1[len1-i-1]-ord('0'))
		nu2 = ord(num2[len2-j-1]-ord('0'))
		result+=(nu1*10**i)*(nu2*10**j)

        return str(result)
```
